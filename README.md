# 📅 秋招计划表 (qiuzhao-planner)

一个纯前端、单文件 HTML 的秋招个人计划表。浏览器打开即用，数据本地保存，无需后端、无需联网。

整合了秋招求职最需要管理的三件事：

- 💼 **面试时间** —— 日历视图管理笔试/面试时间安排
- 📚 **准备计划** —— 周视图按天填写项目 / 八股 / 算法准备内容，带进度条
- 📮 **投递记录** —— 列表记录投递公司、岗位、状态，支持搜索筛选

## ✨ 特性

### 面试时间
- 月历视图，今天高亮，周末标红
- 每格显示前 2 条事件彩条（按类型分色）
- 未来 3 天提醒卡片，临近事件红色高亮
- 事件字段：公司、类型（笔试/初面/二面/HR面/其他）、日期、时间、时长、形式、地点/链接、准备状态、备注

### 准备计划
- 近 3 天视图（今天 + 未来 2 天），三列横排
- 每天分三组：💻 项目准备 / 📖 八股复习 / ⚡ 算法刷题
- 内联编辑：点 + 添加输入回车保存，点击文本可编辑，勾选完成划线
- 每条带进度条，点击调整 0–100%
- 今日高亮，显示当日完成进度

### 投递记录
- 统计卡片：总投递 / 进行中 / Offer / 已结束
- 按投递日期倒序列表
- 状态筛选（已投递/测评/笔试/面试中/Offer/拒绝/终止）+ 公司岗位搜索
- 字段：公司、岗位、投递日期、状态、渠道、城市、备注

### 通用
- 纯前端，单文件 `index.html`，零依赖
- 数据存 `localStorage`，刷新重启不丢
- 三套数据独立存储，互不干扰
- 响应式，手机也能用
- Esc 关闭抽屉、点击遮罩关闭、自动聚焦输入框

## 🚀 使用

直接双击 `index.html` 用浏览器打开即可。

> 数据存在浏览器本地 `localStorage` 中。换浏览器或清缓存会丢失，建议定期手动备份（可自行加导出功能）。

## 🛠 技术栈

原生 HTML + CSS + JavaScript，无任何框架和构建工具。单文件交付，便于分享和部署（可直接托管到 GitHub Pages）。
